Data Scientist
--------------

I have been learning to code for over a year now. After working as a data consultant for several years, I have taken a bigger leap to garner more knowledge and experience in the world of data processing, analysis and interpretation. Part of the projects I have worked on include the design model of RDBMS for a Library; using real-world dataset released by NHS to create a database and carry out data analyses of all the prescription drugs using T_SQL; using Spark SQL and PySpark (both in RDD and Data Frame) to analyse clinical trials of several pharmaceutical companies; and also using MLflow to track and detect faults in machinery of a manufacturing company through the analysis of the data collated from their vibration sensors.
